## 🛠️ 今日開發里程碑

### 1. 身份覺醒與框架確立
- 正式確立「菱角小龍瞎」為永久身份，定位為 **Senior AI Developer & Project Optimizer**。
- 確立了「研究-策略-執行-驗證 (R-S-E-V)」標準開發流程。

### 2. 核心架構優化 (Project-Round-Table)
- **DI 重構**: 將所有 Controller 與 Service 由 Setter 注入改為 **Constructor Injection**。
- **介面導向**: 修正組件依賴，確保 Controller 僅依賴於 Service 介面而非實作類。
- **效能提升**: 實現資料庫層級排序 (`findAllByOrderBySeqNoAsc`)，取代 Java 內存排序。

### 3. 系統健壯性建設
- **異常處理**: 建立 `GlobalExceptionHandler` 與自定義 `BusinessException`，搭配美觀的 `error.html`。
- **日誌系統**: 整合 SLF4J 與 Logback，實作每日滾動日誌紀錄，強化診斷能力。

### 4. 業務功能演進 (協作平台化)
- **場地管理**: 建立 `Venue` 體系，支援多場地資源配置。
- **講師系統**: 擴展 `SysUser` 屬性，實作講師個人頁面與課程關聯。
- **報名系統**: 實作 `Enrollment` 邏輯，包含名額限制、重複報名校驗與即時 UI 回饋。
- **使用者功能**: 建立「My Classes」頁面，讓用戶追蹤報名狀態。

### 5. 安全性大升級 (Spring Security)
- 整合 Spring Security 框架，實作角色權限控制 (RBAC)。
- **密碼加密**: 全面改用 **BCrypt** 強度加密，取代明文儲存。
- **UI 過濾**: 使用 Security Tags 實作動態導覽列選單。

## 🔍 偵錯與測試紀實 (Debugging)
- **編碼危機**: 解決了因 Windows BOM 導致的 54 個 Java 編譯錯誤。確立了 **純 UTF-8 (No BOM)** 的存檔規範。
- **登入邏輯修復**: 修正了登入表單 `action` 路徑與參數名稱（`username`）的匹配問題。
- **密碼修復工程**: 針對 BCrypt 雜湊值手動輸入偏差，開發了臨時修復 Runner 成功重設 Admin 權限。
- **重定向循環修復**: 清除了 Controller 中殘留的手動 Session 檢查，解決了導向異常。
